- ID
- Secret Code
- Name
- Description

Test table:
- Teacher ID
- Test ID (6 digits, letters and numbers, randomly generated)
- Description
- preferred language

Words table
- Test ID
- Word ID
- Word
- Sentence

Answers table
- Test ID
- Word ID
