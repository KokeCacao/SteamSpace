Steam Space connects teachers with programmers to build educational applications. Teachers know what is needed, programmers can deliver it.

This interaction is not unstructured however. We believe in building in the teacher-student interaction from the start, using existing tools that teachers are comfortable with. Our framework facilitates this.

    Our current tool-kit is:
  • Chrome apps for students, and
  • Google Docs/Google Drive for teachers.

Get involved!



What is the Steam Space Framework?

The Steam Space Framework provides real-time feedback from students to teachers by integrating chrome apps with the teachers' Google Docs.

Each Steam Space app has its own look and feel appropriate for the lesson it offers. Some apps run in stand-alone mode and can be downloaded right now and used without needing a teacher's involvement. However the Steam Space framework offers additional value and allows for a teacher to be involved with the student.

How Does It Work?

  • Everything is accomplished via a few docs in the teacher's Google Drive.
  • The teacher groups students into rosters in a spreadsheet called 'Rosters'.
  • The teacher gives an assignment to a roster by entering a single line into a file called 'Assignments'.
  • Output from the assignment goes into its own spreadsheet. Data is updated immediately as the student uses the app.
